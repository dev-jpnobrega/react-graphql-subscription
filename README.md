# React + Apollo + GraphQL + Subscription 🚀

## 💾 Installation Client

```bash
cd client/app

yarn install
```

## 💾 Installation Server

```bash
cd server

yarn install
```

## ▶️ Usage

# Start server
App start http://localhost:8082

```bash
cd server

yarn start
```

# Start client
App start http://localhost:3000

```bash
cd client/app

yarn start
```
